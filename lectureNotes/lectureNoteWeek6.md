**Usability Test**  
- Setting goals and measurements
	- What do you want to learn from the test
	- how do you want to measure the results
		- Time on task
		- Percentage of task completed 
		- Number of clicks etc.
- Establish user profile
- Select tasks to test 
	- First impressions
	- First tasks
	- Critical Tasks
	- Specific problem areas
	- New tasks added to a product

- Write a script (You have to repeat the same words in a same manner)
- prepare test environment 
- pilot!
